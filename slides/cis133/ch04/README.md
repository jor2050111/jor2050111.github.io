# CIS133 Chapter 4: CSS Foundations

## Communication job

By the end, CIS133 students should connect one external stylesheet, aim
selectors by role, and validate CSS because one well-structured file can
restyle a whole site without changing its meaning.

## Teaching plan

- Course level: CIS133, with scaffolded first exposure to CSS
- Prior knowledge: HTML structure, attributes, semantic landmarks, and HTML validation
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: The club site works, but browser defaults still make the design choices
- Evidence and practice: Rule anatomy, connection reach, selector scope, color systems, and validator clues
- Final application: Skills Lab 4A, one stylesheet across three pages with zero validator messages

- Palette: `twilight-data`
- Treatment: Cascade laboratory with graph paper, layered rule sheets, selector scopes, and color specimens
