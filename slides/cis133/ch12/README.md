# CIS133 Chapter 12: Scripting Concepts and Publishing

## Communication job

By the end, CIS133 students should read client-side scripts and say
what runs where, explain the server's half of the web in concept, and
run the eight-check pre-flight before publishing the complete six-page
club site to a live https address over secure transfer.

## Teaching plan

- Course level: CIS133, closing Part IV and the course
- Prior knowledge: the client-server model, the club site through Skills Lab 11A, the checking stack, and the device-mode habit
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: The officers ask the question every stakeholder saves for last: "It works on your machine. When can everyone see it?"
- Evidence and practice: The three layers, the one-line script read left to right, the drawn-lock trap, receive-recheck-store, the secure transfer family, the move test, index.html, and the eight-check pre-flight
- Final application: Skills Lab 12A, the home page built and the whole site shipped and verified live
