CIS133 · Width accounting
Compute the visible box

Why does a 400px box render at 450px?

width: 400px;
padding: 20px;
border: 5px solid;
400 + 40 + 10 = 450px. The default width measures content. Padding and border grow outside it.
CIS133 · box-sizing
End the width surprise

Make width mean the whole visible box.

* {
  box-sizing: border-box;
}

Padding now carves inward instead of growing outward.
