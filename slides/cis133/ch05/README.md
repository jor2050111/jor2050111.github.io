# CIS133 Chapter 5: The Box Model and Typography

## Communication job

By the end, CIS133 students should diagnose and control space and type
because readable pages come from measurable box and typography decisions.

## Teaching plan

- Course level: CIS133, with first exposure to CSS spacing and type
- Prior knowledge: CSS rules, selectors, external stylesheets, and validation
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: The club site's colors work, but every region feels cramped
- Evidence and practice: Box layers, DevTools, shorthand, width accounting, font stacks, and readability checks
- Final application: Skills Lab 5A, a spaced and readable three-page club site

## Visual system

- Palette: `canvas-light`
- Treatment: Measurement bench with ruled paper, nested specimens, rulers, and type samples
