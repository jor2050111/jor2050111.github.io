# CIS133 Chapter 11: HTML Forms

## Communication job

By the end, CIS133 students should construct labeled form controls for
every shape of answer, trace submitted data as name-value pairs through
GET and the query string, and design the club's join page as an
accessible form that collects only what it needs.

## Teaching plan

- Course level: CIS133, second chapter of Part IV, The Promised Pages
- Prior knowledge: HTML structure, CSS selectors and states, the Level AA standard, and the club site through Skills Lab 10A
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: Half the paper sign-up scraps are gone by the next meeting, and the officers want the site to do the collecting
- Evidence and practice: The label contract, the input family, the group-is-the-name experiment, the untouched-submit trap, GET and POST, the missing mail carrier, the placeholder law, and the collection rule
- Final application: Skills Lab 11A, the join page live in every page's navigation with a written refusal defense
