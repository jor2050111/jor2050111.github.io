CIS133 · 3.1
One file, three readers

Pixels serve one reader. Semantics serve all three.

  • Browser: renders the page for sighted visitors.
  • Assistive technology: announces the structure.
  • Software: parses meaning without drawing the page.
CIS133 · 3.1
Meaning inside a sentence

Use <strong> for importance and <em> for spoken stress.

<p>Drop-offs end <em>this</em> Friday.</p>

Stress changes which Friday the sentence means.

<strong>Back up your data before the reset.</strong>

Importance marks the warning visitors must not miss.

If removing the markup changes the meaning, choose a semantic element.
CIS133 · 3.1
Use the blank labels last

<div> and <span> group content without naming it.

Block

<div>

A generic block-level container that stacks like a brick.

Inline

<span>

A generic inline container that flows inside a sentence.

Professional habit

Reach for a semantic element first. Use a generic container when no meaningful element fits.

Organize inside main

Ask whether the content can stand alone.

Yes

No

<section>

Neither

<div>

CIS133 · 3.3
Embed the file

The <img> element works through attributes.

<img
    src="images/cactus-garden.png"
    alt="Three green cacti in a gravel garden under an orange sky"
    width="400"
    height="400">
srcAddresses the file
altStands in for pixels
width + heightReserve honest space

CIS133 · 3.4
Two families, one zoom test

Raster stores pixels. Vector stores instructions.

Raster · a grid of pixels
zoom 4x

Scale it past its true size and the squares show.

Vector · shapes as math
zoom 4x

The computer redraws it sharp at any size.

Photos are raster by nature. Logos and diagrams live as vectors.

CIS133 · 3.4
Choose a format by trade-off

Content and properties decide the file type.

PNGRaster · Lossless

Flat illustrations, screenshots, and transparency

club-logo.png keeps its see-through corners
JPEGRaster · Lossy

Photographs at a smaller file size

the fall drive photo, at a fraction of the bytes
GIFRaster · 256 colors

Simple frame animation

the three-second sign-up demo
SVGVector · Scalable

Logos, icons, and diagrams

the logo redrawn sharp on a poster
