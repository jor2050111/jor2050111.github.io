# CIS133 Chapter 10: HTML Tables

## Communication job

By the end, CIS133 students should tell table data from lists, prose,
and layout with the two-axes test, build grids with captions, headers,
and scope that every tool can read, and publish the club's events page
as a styled schedule that survives a phone.

## Teaching plan

- Course level: CIS133, opening Part IV, The Promised Pages
- Prior knowledge: semantic structure, CSS selectors and the box model, pseudo-classes, responsive habits, and the Level AA standard
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: A clubmate pasted the fall schedule in as six paragraphs, and nobody could find the workshop
- Evidence and practice: The two-axes test, the layout-table prohibition, the position promise, the silent tbody repair, table anatomy with scope, the stripe count trap, and the phone floor
- Final application: Skills Lab 10A, the events page live in every page's navigation
