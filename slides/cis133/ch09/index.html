CIS133 · Heading order
The outline Devon hears

Skipped levels tear pages out of the book.

<!-- BROKEN: the outline skips a level -->
<h1>Campus Tutoring Center</h1>
<h2>Math Help</h2>
<h4>Drop-in Hours</h4>
<!-- REPAIRED: each step down is one level -->
<h1>Campus Tutoring Center</h1>
<h2>Math Help</h2>
<h3>Drop-in Hours</h3>

Want the smaller size? Keep the level. Set the size in the stylesheet.

CIS133 · The keyboard walk
Criterion 2.4.7 · Focus Visible

Put the mouse away. The page tells on itself.

/* BROKEN: keyboard visitors go blind */
.study-links a {
  outline: none;
}
/* REPAIRED: replace the signal, never delete it */
.study-links a:focus {
  outline: 3px solid #268080;
}

Reachable · sensible order · visible at every stop. Under one minute, no installs.

CIS133 · Retrieval check
Trust eyes or math?

Gray #777777 on white measures 4.48 to 1.

Call it: does the pairing pass for body text? For large text?

Body text: fails, 4.48 sits under the 4.5 bar · Large text: passes the 3 to 1 bar · Near-misses fool nearly everyone. Measure, never eyeball.
CIS133 · Color and signal
The sibling rule

Color decorates a signal. It never carries one alone.

<!-- BROKEN: the red is the only signal -->
<p>Sessions shown in red are full.</p>
<li class="session-full">Algebra, Tuesday 3:00</li>
<!-- REPAIRED: the label carries it, the red decorates -->
<li class="session-full">Algebra, Tuesday 3:00 (full)</li>

The repair removes nothing. In-sentence links keep their underline for the same reason.

CIS133 · px and rem
Two promises

px freezes the type. rem respects the visitor.

body {
  font-size: 1rem;    /* was 16px: 16 / 16 */
}
.center-heading {
  font-size: 1.5rem;  /* was 24px: 24 / 16 */
}
CIS133 · Retrieval check
One division

Convert a 32px heading to rem.

Show the math, and name whose setting the converted value respects.

32 / 16 = 2rem · It respects the visitor's own browser default text size, raised once and honored everywhere
CIS133 · Four alt jobs
The decision table

Alt text follows the image's job on this page.

Informative

Describe what matters, about a sentence

Decorative

alt="", empty on purpose, tools skip it

Functional

Name the destination, never the picture

Complex

Deliver the finding, caption beside it

The job belongs to the page, not the file. No alt decision survives a copy and paste unexamined.

CIS133 · The audit stack
Tools, then humans

Tools catch absence. Humans judge honesty.

  • HTML validator: missing alt, skipped levels
  • Contrast checker: one pairing, graded
  • WAVE: a whole rendered page, flagged
  • Seven human checks: truth, order, zoom
CIS133 · Triage
Audit first, repair second

A findings list is not a repair order until you rank it.

Blockerslock visitors out: missing alt, a walk that cannot finish, failing body contrast

Polishdegrades the visit: a wordy decorative alt, a label that could read better

Repair blockers first, and defend the order out loud. A stakeholder will ask.
