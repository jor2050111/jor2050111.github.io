# CIS133 Chapter 8: Responsive Design and Media Queries

## Communication job

By the end, CIS133 students should put the viewport meta tag in every
head, write media queries at breakpoints their own content chooses, and
organize a stylesheet as base styles plus overrides so one file serves
every screen.

## Teaching plan

- Course level: CIS133, second chapter of Part III, Design for Every User
- Prior knowledge: the box model, Flexbox, the club stylesheet, and Lab 7A's phone wireframes
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: The phone wireframes made promises the toolbox cannot keep yet
- Evidence and practice: The bending inventory, the make-believe canvas, query anatomy, boundary behavior, the 584 arithmetic, and the source order law
- Final application: Skills Lab 8A, retrofitting the club site with a content-driven query block proven at three widths
