# CIS133 Chapter 6: Layout with Flexbox

## Communication job

By the end, CIS133 students should arrange and scope an accessible layout
because Flexbox and relationship selectors turn semantic boxes into a usable site.

## Teaching plan

- Course level: CIS133, with first exposure to CSS layout
- Prior knowledge: Semantic HTML, selectors, the box model, and typography
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: The club site's boxes look right, but they still stack
- Evidence and practice: Flex roles, axes, spacing ownership, wrapping, selector scope, and equal states
- Final application: Skills Lab 6A, the Part II milestone with an accessible Flexbox navigation bar
