# CIS133 Chapter 7: UX and Web Design

## Communication job

By the end, CIS133 students should separate UX from UI with five
principles, draw a sitemap and wireframes before writing code, and
assemble a six-section site plan a stakeholder can approve.

## Teaching plan

- Course level: CIS133, opening Part III, Design for Every User
- Prior knowledge: semantic structure, CSS, the box model, and Flexbox
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: The officers want three more pages, and a plan before anyone writes markup
- Evidence and practice: Cost arithmetic, three planning questions, the five principles, sitemap and wireframe rules, the label law
- Final application: Skills Lab 7A, a stakeholder-ready site plan for the six-page club site
