# CIS215 Chapter 12 Presentation

This 20-slide deck teaches ethical analysis, reproducible workflows, Git,
portfolio evidence, and continued learning through a transparent-ledger
motif. It enacts bias mechanisms, the 311 reporting gap, data minimization,
project structure, a handoff test, Git states, repository boundaries, and a
portfolio project trace.

Palmer Penguins values were recomputed from the Chapter 12 course-data file.
Presenter notes supply a 55-minute teaching path.

Open `index.html` through the repository's local server. Use the arrow keys to
move and press `N` to open presenter notes.
