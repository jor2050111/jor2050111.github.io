# CIS215 Chapter 6: Data Cleaning and Preparation

## Communication job

By the end, CIS215 students should audit, repair, and document a messy dataset
because every cleaning choice changes the evidence and who remains represented.

## Teaching plan

- Course level: introductory data analytics after exploration and summary statistics
- Prior knowledge: DataFrame inspection, statistics, grouping, and filtering
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: a specified cleaning workflow reduces 80 survey rows to 67
- Evidence and practice: missingness profiles, treatment choices, type conversion,
  duplicate policy, text normalization, repair order, and a cleaning log
- Final application: Skills Lab 6A, Cleaning Customer Survey Data

## Visual system

- Palette: `canyon-sand`
- Treatment: data quality audit bench with tickets, repair marks, before-and-after
  comparisons, and documented decisions
- Content source: `chapter-06.md` in the CIS215 textbook repository
- Data source: fixed-seed synthetic survey dataset with fictional names in the
  course data pack
