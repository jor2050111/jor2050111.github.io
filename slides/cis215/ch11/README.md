# CIS215 Chapter 11 Presentation

This 20-slide deck teaches external data acquisition through a data-port
motif. It enacts file parsing, workbook sheets, nested JSON, API response
boundaries, credentials, ethical scraping checks, HTML tables, SQL
pushdown, and source selection.

All Skills Lab values were recomputed from the Chapter 11 course-data
files. Presenter notes supply a 55-minute teaching path.

Open `index.html` through the repository's local server. Use the arrow
keys to move and press `N` to open presenter notes.
