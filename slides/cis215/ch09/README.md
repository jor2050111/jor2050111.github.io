# CIS215 Chapter 9 Presentation

This 20-slide deck teaches introductory statistical analysis through an
evidence-lab motif. It enacts correlation, hypothesis logic, p-value
interpretation, t-tests, chi-square tests, error trade-offs, and claim
boundaries.

All Skills Lab values were recomputed from the Chapter 9 synthetic
course-data file. Presenter notes supply a 55-minute teaching path.

Open `index.html` through the repository's local server. Use the arrow
keys to move and press `N` to open presenter notes.
