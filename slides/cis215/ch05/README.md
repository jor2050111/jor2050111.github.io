# CIS215 Chapter 5: Data Exploration and Summary Statistics

## Communication job

By the end, CIS215 students should inspect, summarize, and group a dataset
because one average can hide the shape that changes a decision.

## Teaching plan

- Course level: introductory data analytics after DataFrame selection and filtering
- Prior knowledge: pandas DataFrame creation, selection, filtering, and columns
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: mean price and median price tell different stories
- Evidence and practice: first-look protocol, center, spread, skew, grouping,
  aggregation, sorting, and top-N analysis
- Final application: Skills Lab 5A, Analyzing Sales Performance Data

## Visual system

- Palette: `midnight-gold`
- Treatment: distribution field notes with rulers, dot plots, grouped bars, and
  ranked evidence
- Content source: `chapter-05.md` in the CIS215 textbook repository
- Data source: fixed-seed synthetic e-commerce dataset in the course data pack
