# CIS215 Chapter 2: Working with JupyterLab

Open `index.html` in a modern browser or use a local web server.

The completed deck uses a warm workshop-bench treatment and the
`golden-hour` palette. All teaching figures (window mockups, notebook
cells, keycaps, the traceback, and the Markdown pair) are built in
HTML and CSS, so the deck needs no image assets. Run the validator
from the repository root:

```bash
./scripts/validate-decks
```
