# CIS215 Chapter 4: Working with pandas DataFrames

## Communication job

By the end, CIS215 students should create, select, and filter DataFrames
because a precise subset turns a raw table into answerable evidence.

## Teaching plan

- Course level: introductory data analytics with prior Python container skills
- Prior knowledge: lists, dictionaries, conditionals, loops, and JupyterLab
- Delivery: 55-minute instructor-led class with review value after class
- Opening problem: a 60-row city table is too large to inspect one row at a time
- Evidence and practice: DataFrame anatomy, creation routes, label and position
  selection, boolean masks, query syntax, and calculated columns
- Final application: Skills Lab 4A, City Population Analysis

## Visual system

- Palette: `twilight-data`
- Treatment: selection lens with grid fields, row masks, brackets, and focused cells
- Content source: `chapter-04.md` in the CIS215 textbook repository
- Data source: course data pack city dataset, derived from plotly datasets and US
  Census population estimates
