# CIS215 Chapter 3: Data Structures and Control Flow

Open `index.html` in a modern browser or use a local web server.

The completed deck uses a midnight bracket-blueprint treatment and the
`midnight-gold` palette. All teaching figures (the index ruler, the
slice window, the key-value map, and the decision ladder) are built in
HTML and CSS, so the deck needs no image assets. Run the validator
from the repository root:

```bash
./scripts/validate-decks
```
