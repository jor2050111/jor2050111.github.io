# CIS215 Chapter 8 Presentation

This 20-slide deck teaches advanced pandas manipulation through a
transformation-workbench motif. Its enacted mechanisms cover sorting,
ranking, pivot tables, cross-tabulation, merges, concatenation, and
wide-to-long reshaping.

All values shown from the Skills Lab data were traced to the Chapter 8
course-data files. Presenter notes supply a 55-minute teaching path.

Open `index.html` through the repository's local server. Use the arrow
keys to move and press `N` to open presenter notes.
